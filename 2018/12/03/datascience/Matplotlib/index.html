Matplotlib

发表于 2018-12-03 分类于 教程

Matplotlib 是 Python 的绘图库。 它可与 NumPy 一起使用，提供了一种有效的 MatLab 开源替代方案。使用它可以很方便的绘制出版质量级别的图形。

matplotlib提供了方法用于处理简单动画的绘制。定义update函数用于即时更新图像。
